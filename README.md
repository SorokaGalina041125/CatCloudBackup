# CatCloudBackup

Программа для резервного копирования картинок кошек с сайта [cataas.com](https://cataas.com) в облачное хранилище Яндекс.Диск.

## Описание

Проект позволяет:

- Получать картинки кошек с текстом через API cataas.com
- Загружать их в Яндекс.Диск в папку
- Сохранять информацию о загруженных файлах в JSON-файл

## Установка и настройка

### 1. Установка зависимостей

Установите необходимые зависимости проекта:

```bash
pip install -r requirements.txt
```

### 2. Получение токена Яндекс.Диска

Для работы программы необходимо получить OAuth токен Яндекс.Диска:

1. Перейдите на страницу [Яндекс.Диск Полигон](https://yandex.ru/dev/disk/poligon/)
2. Авторизуйтесь с помощью вашего аккаунта Яндекс
3. Получите OAuth токен для доступа к API Яндекс.Диска

### 3. Настройка переменных окружения

Создайте в корне проекта файл `.env` и добавьте в него полученный токен:

```env
YANDEX_TOKEN=ваш_токен_здесь
```

**Пример заполнения файла `.env`:**

```env
YANDEX_TOKEN=y0_AgAAAAA1234567890abcdefghijklmnopqrstuvwxyz
```

**Важно:**

> - Файл `.env` должен находиться в корне проекта
> - Не добавляйте файл `.env` в систему контроля версий (добавьте его в `.gitignore`)
> - Замените `ваш_токен_здесь` на реальный токен, полученный с сайта Яндекс.Диск Полигон

## Использование

После настройки токена запустите программу:

```bash
python main.py
```
